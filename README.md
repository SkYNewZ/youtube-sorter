# youtube-sorter
Sort videos from YouTube playlist

## Usage

```shell
Usage of youtube-sorter:
      --cache-dir string                 Custom cache directory
      --client-credentials-file string   Google Developers Console client_credentials.json file (default "client_credentials.json")
      --dry-run                          Only show sort without sorting playlist
      --interval duration                Run sort process at this interval (default 12h0m0s)
      --log-level Level                  Log level (default info)
      --playlist string                  Which playlist to sort
      --reverse                          Sort videos by duration in reverse order

```
